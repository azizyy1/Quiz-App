# Quiz App

Quiz App est une application interactive de quiz permettant aux utilisateurs de répondre à une série de questions et de consulter leur score final.  
Le projet a été réalisé dans le but de pratiquer la logique de programmation, la gestion des questions/réponses et l’interaction avec l’utilisateur.

## Description

Cette application propose une expérience simple et intuitive pour tester ses connaissances à travers un quiz.  
L’utilisateur peut lire les questions, choisir une réponse parmi plusieurs propositions, puis obtenir un résultat à la fin du quiz.

Ce projet m’a permis de renforcer mes compétences en développement, notamment dans la manipulation des données, la gestion des événements et l’organisation d’une application interactive.

## Fonctionnalités

- Affichage des questions du quiz
- Choix de réponses multiples
- Vérification des réponses sélectionnées
- Calcul automatique du score
- Affichage du résultat final
- Interface simple et facile à utiliser

## Technologies utilisées

- HTML
- CSS
- JavaScript

## Objectifs du projet

- Créer une application interactive
- Améliorer la logique de programmation
- Gérer les événements utilisateur
- Manipuler les données d’un quiz
- Développer une interface claire et responsive
